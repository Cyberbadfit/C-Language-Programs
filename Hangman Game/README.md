# 🧠 GAME CONCEPT

The game hides a secret word.

Example:

```text
_ _ _ _ _ _
```

You must guess the hidden letters before the Hangman is completed.

Every wrong guess increases danger ☠️

# 🎮 HOW TO PLAY

## Step 1️⃣

Choose difficulty:

```text
1. Easy
2. Medium
3. Hard
```

---

## Step 2️⃣

Guess letters.

Example:

```text
Enter Letter: a
```

---

## Step 3️⃣

Correct guesses reveal letters:

```text
a _ _ _ _ _
```

---

## Step 4️⃣

Wrong guesses increase Hangman body ☠️

```text
 ______
 |    |
 |    O
 |   /|\
_|___/ \
```

# 🤖 AI HINT SYSTEM

Need help?

Use:

```text
2. Use AI Hint
```

The AI reveals one hidden character 🔥

Example:

```text
[AI HINT] Revealed Letter: r
```

⚠️ Hints reduce your score.
